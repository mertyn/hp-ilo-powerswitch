# hp-ilo-powerswitch
simple power control for hp integrated lights out through a web browser

## configuration
set these environment variables or create a `.env`
```
PORT=5000
USER="<username>"
PASS="<password>"
ILOHOST="<ip-of-ilo>"
ILOUSER="<ilo-user>"
ILOPASS="<ilo-password>"
```
